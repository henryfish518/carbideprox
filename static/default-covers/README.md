## 图片建议

- 尺寸：建议 400x300 像素或 16:9 比例
- 格式：JPG 或 PNG
- 大小：每张图片不超过 100KB
- 内容：与电商、运营、直播、店群等相关

## 图片获取方式

可以从以下免费图片网站获取：
- Unsplash (https://unsplash.com/)
- Pexels (https://www.pexels.com/)
- Pixabay (https://pixabay.com/)

搜索关键词：电商, 营销, 直播, 运营, 电商运营 等
